
# Katy's E-Commerce Store 🛍️

A fully-featured Django e-commerce web application featuring user accounts, a product catalog, shopping cart, order tracking, and custom statuses.

## Features ✨
* **Product Catalog & Search:** Browse products across multiple categories with search and sorting filters.
* **Shopping Cart:** Add, remove, and manage items seamlessly.
* **Order Management & Tracking:** Track order status ("Paid", Shipped, Delivered) and view order history.
* **Admin Dashboard:** Fully configured Django admin panel to manage products, categories, and orders.
